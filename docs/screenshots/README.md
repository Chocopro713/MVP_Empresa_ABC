# Carpeta para Screenshots

Esta carpeta contendrá las capturas de pantalla del sistema:

- `login.png` - Pantalla de inicio de sesión
- `dashboard-admin.png` - Dashboard con rol Admin
- `dashboard-user.png` - Dashboard con rol Usuario
- `dark-mode.png` - Sistema en modo oscuro
- `users-list.png` - Lista de usuarios
- `posts-list.png` - Lista de publicaciones
- `todos-list.png` - Lista de tareas
- `admin-panel.png` - Panel de administración

## Cómo generar capturas

1. Ejecutar el sistema con `docker-compose up`
2. Navegar a http://localhost:4200
3. Usar las credenciales de prueba
4. Capturar las pantallas relevantes
